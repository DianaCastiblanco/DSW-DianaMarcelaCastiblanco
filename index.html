Introducción del Sitio Web

Bienvenido a este espacio en donde encontraras el significado de que es HTML5 sus usos mediante etiquetas, atributos y valores como usarlos y su funcionalidad de esta manera conoceremos como realizar páginas web. El OVI diseño de sitios web está elaborado con el fin de dar a conocer información básica de cómo realizar el diseño y construcción de páginas web. Esto se permitirá conocer por medio de lecturas videos esto permitirá al estudiante interactuar de una mejor manera. El objetivo general del sitio web es, Diseñar un Ovi para que los estudiantes y visitantes al sitio información básica de la estructura básica de html5 y el significado de estructuras, atributos y valores.

Objetivo General

El objetivo general del sitio web es, Diseñar un Ovi para que los estudiantes y visitantes al sitio información básica de la estructura básica de html5 y el significado de estructuras, atributos y valores.

Objetivos especificos

  • Ofrecer información para que el estudiante pueda visualizar la estructura básica de Html5

  • Brindar la documentacion para saber el significado de etiquetas, atributos y valores de esta manera visualizar su utilización.

  • Conocer las diferentes etiquetas y sus usos en el codigi de html5

  • Visualizar los archivos multimedia para ayuda en el proceso de conocimiento
